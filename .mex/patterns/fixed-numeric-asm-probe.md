# Fixed Numeric ASM Probe

Use when C repeatedly probes a short numeric or error token inside a scan:

```c
if (p[0] == '4' && p[1] == '6' && (p[2] == '5' || p[2] == '6')) ...
```

A local `__z88dk_fastcall ST_NAKED` helper can be smaller if the call-site C generated by SDCC is branch-heavy:

- `HL` is the candidate pointer.
- Read exactly the same bytes the C expression already read.
- Return boolean or a tiny class in `L`.
- Keep the helper local to the file unless several measured call sites share the exact contract.

Guardrails:

- Do not add length checks unless the old C had them; that is a correctness change, not a pure shrink.
- Do not use this when a generic parser or existing `str_to_u16()` switch measures smaller.
- Always rebuild before accepting; short two-byte probes often grow once call overhead is included.
